# Instrucciones de como usarlo

```
Opciones:
      --help     Muestra ayuda                              [booleano]
      --version  Muestra número de versión                  [booleano]
  -b, --base     Número de tabla a crear          [número] [requerido]
  -l, --listar   Muestra detalles adicionales
                                          [requerido] [defecto: false]
  -h, --hasta    Hasta que número imprimir la tabla           [número]
```
